# Using Alimiyya Bayazan Workbooks

Thank you for downloading the Alimiyya Bayazan Quranic study workbooks! This guide will help you get started.

## ⚠️ CRITICAL: Install Fonts First!

**Before opening any workbook files, you MUST install the included Arabic font.** Without proper font installation, the Arabic text will display in a generic font (Arial/Times New Roman) instead of the beautiful calligraphic script.

### Font Installation Steps:

#### 1. Locate the Font File

In the extracted folder, you'll find one of these font files:
- **Standard mode**: `AlQuran IndoPak by QuranWBW.ttf`
- **Pro Indo-Pak mode**: `AlQuran IndoPak by QuranWBW.ttf`
- **Pro Uthmani mode**: `KFGQPC Uthmanic Script HAFS.otf`

#### 2. Install the Font

**Windows:**
1. Right-click on the `.ttf` file
2. Select "Install" or "Install for all users"
3. Wait for confirmation

**macOS:**
1. Double-click the `.ttf` file
2. Click "Install Font" in Font Book
3. Close Font Book

**Linux:**
1. Copy the `.ttf` file to `~/.fonts/` (create folder if needed)
2. Run: `fc-cache -f -v`
3. Verify: `fc-list | grep -i quran`

#### 3. Close Microsoft Word

**Important:** If Microsoft Word is currently open, close it completely before proceeding.

#### 4. Restart Microsoft Word

**Critical:** You must restart Word after installing fonts for the new fonts to load properly.

#### 5. Open the Workbooks

Now you can open any `.docx` file in the folder. The Arabic text should display in beautiful calligraphic script!

## 🔧 Troubleshooting

### Arabic Text Looks Wrong
**Problem**: Text appears in Arial or Times New Roman instead of calligraphic script

**Solution**: 
1. Verify font is installed (see installation steps above)
2. Restart Microsoft Word completely
3. Reopen the document
4. If still not working, reinstall the font

### Can't See Tashkeel (Vowel Marks)
**Problem**: Arabic text missing diacritical marks

**Solution**: This is a font issue. Ensure you installed the correct font file included in the archive.

### Tables Look Misaligned
**Problem**: Column widths seem off

**Solution**: 
1. Ensure you're viewing in "Print Layout" mode
2. Zoom to 100% or 125% for best viewing
3. If printing, use landscape orientation

### File Won't Open
**Problem**: Word shows compatibility warnings

**Solution**: 
1. Use Microsoft Word 2010 or later
2. Click "Enable Editing" if prompted
3. Update to latest version of Word if possible

## 📖 Learning Resources

### Understanding the Columns

**Word Column**: The actual Quranic word with full tashkeel (vowel marks)

**Root Column**: The 3-letter Arabic root (e.g., ك ت ب for writing-related words)

**Sarf/Role Column** (Pro mode): Morphological form and grammatical role

**Notes Column**: Your space for:
- Translations
- Tafseer notes
- Grammar observations
- Cross-references
- Personal reflections

### Color Coding (Pro Mode)

- **Blue (Nouns)**: Names, places, objects, concepts
- **Red (Verbs)**: Actions, states, commands
- **Grey (Particles)**: Prepositions, conjunctions, articles
